## Automation Notes

This template can be partially automated:

- **Member counts:** Pull from Registry
- **Post counts:** Pull from platform analytics
- **Voting data:** Pull from governance system
- **Ritual attendance:** Pull from event system
- **Moderation counts:** Pull from mod logs

**Manual input required:**
- Context and narrative
- Notable incidents (anonymized)
- Wins and challenges
- Action items

## Historical Archive

Store each monthly report in:
`/archive/reports/[year]-[month].md`

Enable trend analysis by comparing:
- Same month last year
- Previous month
- Rolling 6-month average
